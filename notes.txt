when using redux with react-query, you can dispatch the actions im multiple ways:


1. Inside React-query hooks:

    a. if im using useMutation => then use it inside useMutate in the react-query hooks 
    b. onError (inside react-query hooks)
    c. onSucess (inside react-query hooks)

2. Outside react-query hooks: (compnents)

 ex:   addComment.mutate(
      {
        body: comment,
        post_id: +id,
      },
      {
        onSuccess: () => {
          dispatch(...)
        },
        onSeteled: () => {
            dispatch(...)
        },
        onError: () => {
            dispatch(...)
        }
      }
    );    

3. with the query itself:

ex: const { isLoading, isError, error, data, isStale, refetch } = useGetPosts(
    selectedStatus,
